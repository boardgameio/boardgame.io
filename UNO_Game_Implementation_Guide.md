# UNO 游戏项目实现指南

## 项目概述

这是一个基于 boardgame.io 框架实现的 UNO 卡牌游戏。该项目完整实现了 UNO 的游戏规则，包括特殊牌效果、喊 UNO 机制和多人游戏支持。

## 依赖项

要运行这个项目，你需要安装以下依赖：

```bash
# 核心游戏引擎
npm install boardgame.io

# 用户界面库
npm install react react-dom

# 开发工具（如果需要）
npm install --save-dev parcel-bundler
```

## 项目结构

项目包含以下主要文件：

- `src/games/uno/game.js` - 游戏核心逻辑和规则
- `src/games/uno/board.js` - React 用户界面组件
- `src/games/uno/board.css` - 样式表
- `src/games/uno/index.js` - 游戏入口和配置

## 游戏功能

### 核心游戏逻辑 (game.js)

**卡牌系统：**
- 108 张卡牌（数字牌、功能牌和万能牌）
- 四种颜色：红、蓝、绿、黄
- 特殊牌：跳过、反转、+2、万能牌、+4

**游戏规则：**
- 玩家轮流出牌，必须匹配颜色或数字/类型
- 特殊牌有特殊效果（改变方向、跳过玩家、抽牌等）
- 当玩家只剩一张牌时必须喊"UNO"
- 第一个出完所有牌的玩家获胜

**游戏动作：**
- 出牌 (playCard)
- 选择万能牌颜色 (chooseColor)
- 抽牌 (drawCard)
- 喊 UNO (sayUno)
- 指出别人没喊 UNO (callUno)

**游戏状态管理：**
- 牌堆和弃牌堆
- 玩家手牌
- 当前颜色和方向
- 喊 UNO 状态
- 胜利条件检查

### 用户界面 (board.js)

**游戏界面组件：**
- 卡牌显示
- 玩家手牌区域
- 牌堆和弃牌堆
- 游戏状态指示器
- 颜色选择器
- 玩家信息显示

**交互功能：**
- 点击卡牌出牌
- 点击牌堆抽牌
- 选择万能牌颜色
- 喊 UNO 按钮
- 指出其他玩家未喊 UNO

### 样式 (board.css)

- 卡牌样式和颜色
- 游戏布局
- 交互效果（悬停、动画）
- 响应式设计

### 游戏入口 (index.js)

- 本地游戏客户端配置
- 多人游戏客户端配置
- 游戏大厅界面

## 游戏特性

- **多人游戏支持：** 支持 2-10 名玩家
- **信息隐藏：** 玩家只能看到自己的手牌
- **自动规则执行：** 特殊牌效果自动应用
- **回合管理：** 自动处理玩家回合和方向变化
- **完整的 UNO 规则：** 包括喊 UNO 和罚牌机制

## 扩展可能性

这个项目可以进一步扩展：

- 添加计分系统
- 实现自定义规则选项
- 添加动画效果
- 改进用户界面设计
- 添加音效
- 实现游戏统计和历史记录
- 添加 AI 玩家

## 使用方法

1. 创建上述文件结构
2. 安装必要的依赖
3. 在主应用中导入并使用 UnoLobby 组件
4. 对于在线多人游戏，需要设置 boardgame.io 服务器

## 总结

这个 UNO 游戏项目展示了如何使用 boardgame.io 框架创建一个功能完整的多人回合制游戏。通过分离游戏逻辑和用户界面，项目结构清晰，易于维护和扩展。boardgame.io 框架处理了复杂的游戏状态同步和多人游戏功能，让开发者可以专注于实现游戏规则和用户体验。
